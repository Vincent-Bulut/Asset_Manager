
Pour pré-remplir un formulaire en utilisant une requête GET avec Svelte, vous pouvez suivre cet exemple de mise en œuvre. Cet exemple suppose que vous avez déjà configuré un projet Svelte de base.

1. **Installation des dépendances:**
   Assurez-vous que votre projet Svelte est configuré avec les dépendances nécessaires. Si vous ne l'avez pas déjà fait, initialisez un nouveau projet Svelte :

   ```bash
   npx degit sveltejs/template svelte-app
   cd svelte-app
   npm install
   ```

2. **Création du composant de formulaire:**

   Créez un composant `Form.svelte` où nous allons faire la requête GET et pré-remplir le formulaire.

   ```svelte
   <!-- src/components/Form.svelte -->
   <script>
       import { onMount } from 'svelte';
       import { writable } from 'svelte/store';

       const formData = writable({
           name: '',
           email: '',
           age: ''
       });

       onMount(async () => {
           try {
               const response = await fetch('https://api.example.com/form-data');
               const data = await response.json();
               formData.set({
                   name: data.name,
                   email: data.email,
                   age: data.age
               });
           } catch (error) {
               console.error('Failed to fetch form data:', error);
           }
       });
   </script>

   <style>
       form {
           display: flex;
           flex-direction: column;
           max-width: 300px;
           margin: auto;
       }

       label {
           margin-top: 10px;
       }
   </style>

   <form>
       <label>
           Name:
           <input type="text" bind:value={$formData.name} />
       </label>
       <label>
           Email:
           <input type="email" bind:value={$formData.email} />
       </label>
       <label>
           Age:
           <input type="number" bind:value={$formData.age} />
       </label>
   </form>
   ```

3. **Incorporation du composant dans votre application principale:**

   Ensuite, vous devez inclure le composant `Form.svelte` dans votre application principale.

   ```svelte
   <!-- src/App.svelte -->
   <script>
       import Form from './components/Form.svelte';
   </script>

   <main>
       <h1>Pré-remplir un formulaire avec Svelte</h1>
       <Form />
   </main>

   <style>
       main {
           text-align: center;
           padding: 1em;
           max-width: 240px;
           margin: 0 auto;
       }

       h1 {
           color: #ff3e00;
           text-transform: uppercase;
           font-size: 1.5em;
           margin-bottom: 0.5em;
       }
   </style>
   ```

4. **Exécution de l'application:**

   Lancez votre application en utilisant :

   ```bash
   npm run dev
   ```

   Cela devrait démarrer un serveur de développement et ouvrir votre application dans un navigateur. Votre formulaire sera pré-rempli avec les données récupérées via la requête GET lorsque le composant sera monté.

Ce modèle de base montre comment récupérer des données depuis une API et les utiliser pour pré-remplir un formulaire dans une application Svelte. Vous pouvez adapter l'exemple en fonction de vos besoins spécifiques et ajouter des fonctionnalités supplémentaires selon vos exigences.
